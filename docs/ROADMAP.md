# Roadmap

## v1.0.0 (Current)
- Core Rust + React Architecture
- TrustTier Security Engine
- Workspace Isolation
- OpenRouter Integration
- DPAPI Keychain
- Audit Log UI

## v1.1.0
- **Anthropic Direct API**: Bypass routing layers for direct Claude 3.5 Sonnet access.

## v1.2.0
- **Local Models**: Full integration with Ollama for 100% offline agentic coding.
- **MCP Integration**: Plug-and-play Model Context Protocol servers.

## v2.0.0
- **Docker Sandboxing**: Spin up ephemeral containers for the agent to execute code inside, completely insulating the host machine from accidental deletions.
- **Cloud Sync**: End-to-end encrypted synchronization of Trusted Sessions and chat histories.
